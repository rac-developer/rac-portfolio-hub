## 📁 Estructura del Proyecto

El repositorio está organizado por ecosistemas y tecnologías:

### 🖥️ Web Frontend (React, Next.js & Astro)
Ubicado en `web-frontend/`, esta sección contiene aplicaciones interactivas y optimizadas para SEO.

*   **Next.js Projects**:
    *   **Dashboard Pro**: Un panel de administración avanzado con autenticación (NextAuth v5), validación de formularios (Zod) y componentes optimizados.
    *   **Redux Toolkit Hub**: Implementaciones sólidas de manejo de estado global.
*   **Astro Projects**:
    *   **Minimalist CV**: Plantilla de currículum minimalista optimizada para rendimiento y SEO.
*   **React (Vite) Projects**:
    *   **Disney+ Clone**: Replicación de interfaz con consumo de APIs externas y diseño responsive.
    *   **Kanbas Board**: Aplicación de productividad con funcionalidad Drag & Drop (`react-beautiful-dnd`).
    *   **GSAP Animations**: Demostración de micro-interacciones y animaciones de alto rendimiento.
    *   **Firebase Auth**: Sistema de login y registro integrado con Firebase.

### ⚙️ Logic Backend (NestJS)
Ubicado en `logic-backend/`, enfocado en la arquitectura de servidor y persistencia de datos.

*   **Core API**: Desarrollo de servicios RESTful utilizando NestJS, integración con **Prisma ORM**, validación de datos y documentación automatizada con **Swagger**.

## 🚀 Cómo Explorar los Proyectos

Cada subproyecto es independiente. Para ejecutar cualquiera de ellos:

1.  Clona este repositorio:
    ```bash
    git clone https://github.com/tu-usuario/rac-portfolio-hub.git
    ```
2.  Navega a la carpeta del proyecto deseado:
    ```bash
    cd web-frontend/next/01-dashboard
    ```
3.  Instala las dependencias:
    ```bash
    pnpm install  # o npm install / yarn
    ```
4.  Inicia el servidor de desarrollo:
    ```bash
    pnpm dev
    ```
